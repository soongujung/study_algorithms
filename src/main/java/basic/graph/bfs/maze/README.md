미로에서 0은 벽이고 1은 길이다. (0,0) 에서 시작하여 (2,2) 까지 갈 수 있는 길의 길이를 구해보자. (사실 이게 정확한지는 잘 모르겠다. 다른 책 역시도 찾아보고 공부해야 한다.)
```plain
[1,1,0]
[0,1,0]
[0,1,1]
```
